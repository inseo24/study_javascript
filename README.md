    - 장소 : 비대면 회의
2. 스터디 진행 방식
    - 전 주에 학습량을 정하고 스터디원이 발표할 챕터를 할당 받습니다.

    - 순서에 맞게 발표를 시작하며, 질의 응답 시간을 갖습니다.
    - 발표 시간은 최대 30분으로 제한합니다.
3. 이 레포지터리에는 책과 스터디를 바탕으로 개인적으로 학습한 내용을 재업로드할 예정입니다.
